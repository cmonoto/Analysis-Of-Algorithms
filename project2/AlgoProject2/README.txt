Project 2: Dynamic Programming
================================================================
1.First, open a new Terminal, change the location to the folder
 with your download AlgoProject2.
2.In the command line type: unzip AlgoProject2.zip
3.Go to AlgoProject2 folder. Open input.txt, change the data to 
create your own balls(b) baskets(n) capacity(k). Remember the 
first line in input.txt is how many problems you have!
4.In the command line type: cd AlgoProject2
5.In the command line type: g++ main.cpp
6.In the command line type: ./a.out
7.Open your output.txt file, you will see the solution there.
